# 🚀 PROYECTO: API REST CON NODE.JS & POSTGRESQL

Este documento contiene la guía completa, el código fuente y las instrucciones para montar una API profesional desde cero.

---

## 📂 1. ESTRUCTURA DEL PROYECTO

Crea una carpeta llamada `mi-primera-api` y organiza los archivos de la siguiente manera:

```text
mi-primera-api/
├── node_modules/       # Se crea automáticamente con npm install
├── .env                # Variables sensibles (Base de datos)
├── .gitignore          # Archivos que Git no debe subir
├── index.js            # Código principal de la API
├── package.json        # Configuración y dependencias
└── README.md           # Este documento
/
```
